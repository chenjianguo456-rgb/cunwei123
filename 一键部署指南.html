🌾 村委资料共享管理系统

云端部署 · 24小时在线 · 完全免费

1注册 Render 账号（2分钟）

打开浏览器，访问：

打开 Render.com 注册

点击右上角 Get Started → 选择 Sign up with GitHub 或用邮箱注册

💡 提示：不需要绑定信用卡，免费版完全够用！

2上传代码到 GitHub（1分钟）

在浏览器中操作：

  1. 访问 github.com/new 创建仓库，取名 cunwei
  2. 保持 Public（公开），点击 Create repository
  3. 在创建后的页面，点击 uploading an existing file
  4. 将 整个 cunwei_app 文件夹拖入浏览器窗口
  5. 点击 Commit changes 提交

3 一键部署（1分钟）

回到 Render.com，按以下步骤操作：

  1. 点击顶部 New + → 选择 Web Service
  2. 授权 GitHub → 选择 cunwei 仓库
  3. 关键设置：
    • Name: cunwei
    • Runtime: Python 3
    • Build Command: pip install -r requirements.txt
    • Start Command: python app.py
  4. 往下拉，选择 Free 方案
  5. 点击 Create Web Service

⏳ 等待3-5分钟，部署完成后你会看到一个公网地址：

https://cunwei.onrender.com
