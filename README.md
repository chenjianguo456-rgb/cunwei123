# 🏘️ 村委资料共享管理系统

专为农村基层打造的数字化资料管理平台，支持手机端操作、身份证智能校验、Excel/WPS 兼容导入导出，以及灵活的后台管理。

---

## ✅ 已内置功能

| 功能 | 说明 |
|------|------|
| 乡村振兴 | 项目台账、资金进度管理 |
| 党员信息 | 入党日期、支部归属、年龄性别自动提取 |
| 村民信息 | 常住人口信息库 |
| 养老信息 | 养老保险号、缴费档次管理 |
| 耕地信息 | 地块编号、面积、土地类型管理 |
| 资料台账 | 档案编号、归档日期、保管人 |

**系统特性：**
- 📱 响应式设计，手机/电脑均可访问操作
- 🔢 身份证号码自动校验（前17位加权求和验证末位）
- 👤 性别/年龄根据身份证号自动提取，无需手动录入
- 📊 导出标准 `.xlsx` 文件，可直接用 **WPS表格 / Excel** 打开编辑
- ⚙️ 管理后台可动态增减功能分区、自定义表格模板

## 🚀 快速部署（单机/局域网）

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库并启动

```bash
python app.py
```

首次启动会自动创建 `cunwei.db` 数据库文件，并初始化管理员账户和 6 个默认分区。

### 3. 访问系统

打开浏览器访问：

```
http://localhost:5000
```

同局域网内其他设备访问：

```
http://<你的电脑IP>:5000
```

> 电脑IP可通过命令 `ipconfig` (Windows) 或 `ifconfig` / `ip addr` (Linux/Mac) 查看。

## 🌐 部署到互联网（公网访问）

方案一：使用内网穿透（最简单，适合村级无公网IP场景）
- 推荐工具：花生壳、cpolar、frp、ngrok
- 将本地 5000 端口映射出去，即可通过手机流量或外地网络访问

方案二：租用云服务器（阿里云/腾讯云/华为云）
1. 购买轻量应用服务器（1核2G足够）
2. 服务器安装 Python 3.10+
3. 上传本项目文件到服务器
4. 执行 `pip install -r requirements.txt`
5. 使用生产级服务器启动：
   ```bash
   gunicorn -w 2 -b 0.0.0.0:80 app:app
   ```
6. 绑定域名，配置备案（如需要）

## 📖 使用说明

### 数据录入
1. 首页点击功能分区（如“村民信息”）
2. 点击「添加行」逐条录入
3. 输入身份证号后，**性别和年龄会自动计算填充**
4. 输入错误身份证末位会变红提示
5. 完成后点击「保存」

### 导入已有表格
1. 点击「导入Excel」，选择 `.xlsx` 文件
2. 要求：第一行为表头（与系统模板列名一致），第二行起为数据
3. 导入会覆盖当前分区数据，请提前导出备份

### 导出到 WPS
1. 点击「导出Excel」，下载 `.xlsx` 文件
2. 双击文件可用 **WPS 表格** 直接编辑
3. 编辑后可再导入回系统

### 后台管理
- 登录后右上角点击「管理后台」
- **功能分区**：可增删改分类，修改图标和排序
- **表格模板**：为每个分区设计不同的列（文本、数字、日期、身份证、下拉选择、自动计算）
- **账户管理**：修改管理员密码、添加普通用户（普通用户不能进后台）

## 📞 常见问题

**Q：手机打开页面太小？**  
A：本系统已做手机适配，若仍觉表格小，请横屏查看，或电脑操作。

**Q：数据存在哪里？**  
A：默认保存在程序同目录下的 `cunwei.db` 文件中，请定期备份该文件。

**Q：忘记密码怎么办？**  
A：删除 `cunwei.db` 后重新运行程序，系统会自动重建数据库并恢复默认管理员账户（admin / admin123456），注意这样会清空所有录入数据！

**Q：能多人同时录入吗？**  
A：可以，系统支持多用户同时访问。建议管理员为每位村干部添加普通账户，各自管理不同分区。
