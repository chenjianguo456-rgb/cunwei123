云端部署 · 24小时在线 · 数据永久保存 · 完全免费

1注册 Render 账号（2分钟）

打开 Render.com

点击右上角 Get Started → 用 GitHub 账号注册登录

💡 不需要绑定信用卡，免费版完全够用！

2创建 PostgreSQL 数据库（关键！）

这一步确保数据不会丢失，关机、重启都不影响。

  1. 登录 Render 后，点击 New +
  2. 选择 PostgreSQL
  3. 填写：
    • Database Name: cunwei-db
    • User: 保持默认
    • Region: 选最近的（如 Singapore）
    • Database Version: 16（默认即可）
    • Plan: Free ← 必须选这个！
  4. 点击 Create Database

⏳ 等待1-2分钟创建完成。

创建后，找到 Connections 区域，复制 Internal Database URL（以 postgres:// 开头的那串）

3创建 Web 服务（部署程序）

  1. 回到 Dashboard，点击 New + → Web Service
  2. 选择你刚推送的仓库 cunwei123
  3. 填写设置：
    • Name: cunwei
    • Runtime: Python 3
    • Build Command:
      pip install -r requirements.txt
    • Start Command:
      gunicorn app:app --workers 1 --threads 2 --timeout 120
  4. 添加环境变量（关键步骤！）：
    • 找到 Environment 区域
    • 点击 Add Environment Variable
    • Key 填: DATABASE_URL
    • Value 填: 刚才复制的 PostgreSQL Internal URL
  5. 选择 Free 方案
  6. 点击 Create Web Service

4等待部署完成（3-5分钟）

页面会显示构建日志，等待出现 "Deploy live" 即表示成功。

你会得到一个公网地址：

https://cunwei.onrender.com

❓ 常见问题

Q: 免费版数据库会过期吗？

A: Render 免费 PostgreSQL 有效期90天。到期后可重新创建，或升级到付费版（$7/月）。


Q: 免费版 Web 服务会休眠吗？

A: 15分钟无访问会自动休眠，下次访问时自动唤醒（约30秒）。不影响数据，只是首次打开稍慢。


Q: 数据会丢失吗？

A: 不会！所有数据存在 PostgreSQL 数据库中，图片也存在数据库里。重启、重新部署都不会丢失。
